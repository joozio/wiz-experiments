# ∩ The Conjunction Fallacy

> 8 character vignettes from Linda the philosophy major to Dave the SF vegan. For each, two statements about the person — one simple, one with an extra clause that fits the stereotype. Pick which is more likely. Tversky and Kahneman (1983) found 85% pick the compound. Math says they cannot.
