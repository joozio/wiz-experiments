# ⏳ The Present Bias Test

> 10 trades, 5 hidden pairs. Each pair offers the same wait and the same bonus — once with \"today\" on the table, once with both options pushed years out. A consistent person picks the same way both times. Most people flip. Thaler (1981), Laibson (1997).
